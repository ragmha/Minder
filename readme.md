## 🔧 Prerequisites

* [node](https://nodejs.org/en/)

* [Ionic cli](https://ionicframework.com/docs/cli/)

## 🚀 Getting Started
* ` yarn global add ionic cordova`

* ` git clone https://github.com/ragmha/minder.git`

* ` yarn`

* ` ionic serve[options]`
